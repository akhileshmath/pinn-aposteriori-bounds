\section{Numerical Experiments}\label{sec:experiments}

This section validates the proposed estimator through a series of
reproducible numerical experiments. The objectives are:

\begin{itemize}
\item To verify \emph{reliability}: $\eta \ge 1$,
\item To assess \emph{sharpness}: $\eta \approx 1$,
\item To demonstrate the \emph{failure of residual-only estimators},
\item To quantify the \emph{dominance of boundary mismatch}.
\end{itemize}

All experiments are performed using the implementation described in
Section~\ref{sec:estimator}.

\subsection{Experimental setup}

\paragraph{PINN architecture.}
We employ fully-connected neural networks with $4$ hidden layers and
$64$ neurons per layer, using $\tanh$ activation functions.

\paragraph{Training.}
Optimization is performed using Adam followed by L-BFGS.
The loss weights are fixed to
\[
w_r = 1, \qquad w_b = 10.
\]

\paragraph{Sampling.}
Interior and boundary collocation points are sampled uniformly.
All experiments use sufficiently dense sampling to ensure convergence
of the training loss.

\paragraph{Estimator computation.}
\begin{itemize}
\item Residuals are evaluated using automatic differentiation.
\item The dual problem is solved using DST on structured domains
and FEM on the L-shaped domain.
\item The lifting problem is solved using Laplace harmonic extension.
\end{itemize}

\paragraph{Error measurement.}
The true error is computed using the exact solution:
\[
|u - \hat{u}|_{H^1(\Omega)} = \|\nabla(u - \hat{u})\|_{L^2(\Omega)}.
\]

\paragraph{Reproducibility and evaluation protocol.}
All experiments are executed using the provided code repository,
ensuring full reproducibility of results. The estimator is computed
exclusively from the trained PINN solution and problem data, without
access to the exact solution.
The exact solution is used solely for evaluation of the true error and
effectivity index, and does not enter the estimator computation.

\paragraph{Observations.}

\begin{itemize}

In all cases, $\eta \ge 1$, confirming the theoretical bound.

\item \textbf{Sharpness.}
The estimator remains close to the true error, with
$1.05 \le \eta \le 1.31$.

\item \textbf{Boundary dominance.}
The lifting term is consistently the dominant contribution.
For example, in the Poisson problem:
\[
\text{Residual} \ll \text{Lifting}.
\]

\item \textbf{Singular behavior.}
In the L-shaped domain, the estimator remains reliable despite
reduced regularity.
\end{itemize}

\paragraph{Key findings.}

\begin{itemize}

\item \textbf{Residual-only failure.}
The residual-only estimator underestimates the error by orders of magnitude.
For instance, in the smooth Poisson case:
\[
\eta_r \approx 0.0275,
\]
which is completely unreliable.

\item \textbf{Dominance of lifting.}
The lifting-only estimator is already close to the true error,
indicating that boundary mismatch is the primary source of error.

\item \textbf{Full estimator accuracy.}
The full estimator consistently recovers reliability and achieves
$\eta \approx 1$.

\end{itemize}

These results provide strong evidence that \emph{boundary mismatch is the
dominant error mechanism in PINNs with soft boundary enforcement}.

\subsection{Discussion}

The experiments support the following conclusions:

\begin{enumerate}[leftmargin=*]

\item \textbf{Residual-only estimators are fundamentally insufficient}
for PINNs due to weak enforcement of boundary conditions.

\item \textbf{Boundary mismatch dominates the error} across all
considered problems.

\item \textbf{The proposed estimator is reliable and sharp}, providing
a practical certification tool for PINN solutions.

\item \textbf{Training loss is misleading}, as it fails to reflect
energy norm accuracy.

\end{enumerate}

These findings highlight a critical gap in current PINN practice and
demonstrate the necessity of rigorous a posteriori error estimation.
